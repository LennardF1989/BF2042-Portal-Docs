# ArraySlice

Returns a copy of the specified **Array** containing only values from a specified index range.

### Inputs

1. Array

2. Number

    (Start Index)

3. Number

    (Count)

### Output

-   Array

```blockly
<block type="ArraySlice">
    <value name="VALUE-0">
        <block type="AllPlayers"></block>
    </value>
    <value name="VALUE-1">
        <block type="Number">
            <field name="NUM">0</field>
        </block>
    </value>
    <value name="VALUE-2">
        <block type="Number">
            <field name="NUM">2</field>
        </block>
    </value>
</block>
```
