# LocalToWorldVector

Converts the provided local **Player** vector to the corresponding vector in the world space.

### Inputs

1. Vector

    (Local Vector)

2. Player

### Output

-   Vector

    (World Vector)

```blockly
<block type="WorldVectorOf">
    <value name="VALUE-0">
        <block type="RightVector"></block>
    </value>
    <value name="VALUE-1">
        <block type="EventPlayer"></block>
    </value>
</block>
```
