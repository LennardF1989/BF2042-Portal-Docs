# SkipMandown

Sets the target **Player** to skip the mandown state and go directly to the deploy screen when killed.

### Inputs

1. Player
2. Bool

```blockly
<block type="SkipManDown">
    <value name="VALUE-0">
        <block type="EventPlayer"></block>
    </value>
    <value name="VALUE-1">
        <block type="Boolean">
            <field name="BOOL">TRUE</field>
        </block>
    </value>
</block>
```
