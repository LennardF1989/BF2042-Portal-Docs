# WorldToLocalVector

Converts the provided world vector to the corresponding vector in local **Player** space.

### Inputs

1. Vector

    (World Vector)

2. Player

### Output

-   Vector

    (Local Vector)

```blockly
<block type="LocalVectorOf">
    <value name="VALUE-0">
        <block type="ForwardVector"></block>
    </value>
    <value name="VALUE-1">
        <block type="EventPlayer"></block>
    </value>
</block>
```
