# SetPlayerMaxHealth

Sets the max health of a target **Player**.  
  
_Note: the max health of a player cannot be higher than 1000._

### Inputs

1. Player
2. Number

```blockly
<block type="SetPlayerMaxHealth">
    <value name="VALUE-0">
        <block type="EventPlayer"></block>
    </value>
    <value name="VALUE-1">
        <block type="Number">
            <field name="NUM">500</field>
        </block>
    </value>
</block>
```
