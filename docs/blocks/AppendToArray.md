# AppendToArray

Returns a copy of an **Array** with the provided value appended to the end.

### Inputs

1. Array
2. Any Type

### Output

-   Array

```blockly
<block type="AppendToArray">
    <value name="VALUE-0">
        <block type="EmptyArray"></block>
    </value>
    <value name="VALUE-1">
        <block type="EventOtherPlayer"></block>
    </value>
</block>
```
