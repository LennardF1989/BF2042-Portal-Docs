# SetVariableAtIndex

Finds or initializes an **Array** on a provided **Variable**, and stores a provided value in that **Array** at the specified index.

### Inputs

1. Variable
2. Number

    (Index)

3. Any Type

    (Value)

```blockly
<block type="SetVariableAtIndex">
    <value name="VALUE-0">
    </value>
    <value name="VALUE-1">
        <block type="Number">
            <field name="NUM">0</field>
        </block>
    </value>
    <value name="VALUE-2">
        <block type="EventPlayer"></block>
    </value>
</block>
```
