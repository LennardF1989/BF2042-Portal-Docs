# Heal

Sets the amount of health gained in an instant for a target **Player** applied by a healer **Player**.

### Inputs

1. Player

    (Target)

2. Number
3. Player

    (Healer)

    _Optional_

```blockly
<block type="SetHeal">
    <value name="VALUE-0">
        <block type="EventPlayer"></block>
    </value>
    <value name="VALUE-1">
        <block type="Number">
            <field name="NUM">50</field>
        </block>
    </value>
</block>
```
