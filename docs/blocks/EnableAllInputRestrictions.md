# EnableAllInputRestrictions

Enables or disables all available input restrictions for a target **Player**.

### Inputs

1. Player
2. Bool

```blockly
<block type="EnableAllInputRestrictions">
    <value name="VALUE-0">
        <block type="EventPlayer"></block>
    </value>
    <value name="VALUE-1">
        <block type="Boolean">
            <field name="BOOL">TRUE</field>
        </block>
    </value>
</block>
```
