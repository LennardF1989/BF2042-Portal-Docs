# SpotTarget

Spots a target **Player** for a specified duration of time (in seconds).

### Inputs

1. Player
2. Number

    (Duration)

```blockly
<block type="SpotTarget">
    <value name="VALUE-0">
        <block type="EventPlayer"></block>
    </value>
    <value name="VALUE-1">
        <block type="Number">
            <field name="NUM">10</field>
        </block>
    </value>
</block>
```
