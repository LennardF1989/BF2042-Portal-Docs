# CreateVector

Returns a **Vector** composed of three provided 'X' (left), 'Y' (up), and 'Z' (forward) values.

### Inputs

1. Number

    (X Value)

2. Number

    (Y Value)

3. Number

    (Z Value)

### Output

-   Vector
