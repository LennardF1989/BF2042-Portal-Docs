# PauseGameModeTime

Pauses or unpauses the gamemode timer based on the provided **Bool** input.

### Inputs

1. Bool

```blockly
<block type="PauseRoundTime">
    <value name="VALUE-0">
        <block type="Boolean">
            <field name="BOOL">TRUE</field>
        </block>
    </value>
</block>
```
