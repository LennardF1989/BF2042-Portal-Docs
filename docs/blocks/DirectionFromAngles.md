# DirectionFromAngles

Returns a directional **Vector** from the provided horizontal (yaw) and vertical (pitch) angles (in degrees).

### Inputs

1. Number (Horizontal)
2. Number (Vertical)

### Output

-   Vector

```blockly
    <value name="VALUE-0">
        <block type="Number">

        </block>
    </value>
    <value name="VALUE-1">
        <block type="Number">
            <field name="NUM">130</field>
        </block>
    </value>
</block>
```
