# ValueInArray

Returns the value found at a provided index of an **Array**.

### Inputs

1. Array
2. Number

### Output

-   Any Type

```blockly
<block type="ValueInArray">
    <value name="VALUE-0">
        <block type="AllPlayers"></block>
    </value>
    <value name="VALUE-1">
        <block type="Number">
            <field name="NUM">0</field>
        </block>
    </value>
</block>
```
