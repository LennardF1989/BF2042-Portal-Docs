# RemoveFromArray

Returns a copy of the specified **Array** with the provided value removed.

### Inputs

1. Array
2. Any Type

### Output

-   Array

```blockly
<block type="RemoveFromArray">
    <value name="VALUE-0">
        <block type="AllPlayers"></block>
    </value>
    <value name="VALUE-1">
        <block type="EventPlayer"></block>
    </value>
</block>
```
