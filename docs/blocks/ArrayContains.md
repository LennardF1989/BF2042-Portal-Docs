# ArrayContains

Returns a **Bool** value based on whether the provided **Array** contains the provided value.

### Inputs

1. Array
2. Any Type

### Output

-   Bool

```blockly
<block type="ArrayContains">
    <value name="VALUE-0">
        <block type="AllPlayers"></block>
    </value>
    <value name="VALUE-1">
        <block type="EventPlayer"></block>
    </value>
</block>
```
