# Resupply

Resupplies the target **Player** using a provided **ResupplyTypesItem**.

### Inputs

1. Player
2. ResupplyTypesItem

```blockly
<block type="Resupply">
    <value name="VALUE-0">
        <block type="EventPlayer"></block>
    </value>
    <value name="VALUE-1">
        <block type="ResupplyTypesItem">
            <field name="VALUE-0">ResupplyTypes</field>
            <field name="VALUE-1">SupplyBag</field>
        </block>
    </value>
</block>
```
