# EnablePlayerDeploy

Enables or disables the ability for a target **Player** to deploy.

### Inputs

1. Player
2. Bool

```blockly
<block type="SetSpawnOverride">
    <value name="VALUE-0">
        <block type="EventPlayer"></block>
    </value>
    <value name="VALUE-1">
        <block type="Boolean">
            <field name="BOOL">FALSE</field>
        </block>
    </value>
</block>
```
